# Image Assets

Drop images into this folder. All 5 slots are wired up in index.html via CSS custom properties.

## Required files

| File | Where used | Ideal subject |
|---|---|---|
| `hero.jpg` | Hero right panel | Woman in server room / datacenter, dark lighting, blue light |
| `banner.jpg` | Full-width mid-page banner | Datacenter rows / server racks, dramatic lighting |
| `grid1.jpg` | Photo grid — tall left | Woman working on tech / laptop in dark tech environment |
| `grid2.jpg` | Photo grid — top right | Close-up of server / tech hardware |
| `grid3.jpg` | Photo grid — bottom right | Woman presenting / analysing data on screens |

## Where to get them (free, high quality)

**Unsplash** — https://unsplash.com
Search terms that work well:
- "woman datacenter"
- "woman server room"
- "AI technology woman"
- "tech professional woman blue light"
- "server room dark blue"
- "data center"

Download the largest size available, rename to match the filenames above.

## Tips
- Landscape orientation works best for `banner.jpg`
- Portrait works best for `hero.jpg` and `grid1.jpg`
- Dark images work best — the CSS overlay dims them and adds the on-brand blue tint
- Minimum 1200px wide recommended
